Overview
Description · the ask
Stand up a new AWS development instance and run a pilot demand-management rollout on it.
Business problemThere’s no provisioned AWS landscape to iterate on — setup is manual and slow, holding up the pilot. Every environment spin-up today is a manual ticket through IT, taking 4–6 weeks and blocking the demand-management pilot from starting. A repeatable, governed landing zone would let teams self-serve and shift effort from setup to actual evaluation.
